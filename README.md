# NISETSU — 二節

Single-page landing for **NISETSU**, an atelier producing hand-finished training nunchaku in numbered editions of 120.

See [`BRIEF.md`](BRIEF.md) for the full design brief and handoff spec.

## Local

```bash
npm install
npm start
```

Open http://localhost:3000.

## Deploy

Static site served by [`serve`](https://www.npmjs.com/package/serve). Railway picks up `npm start` automatically; `$PORT` is honoured.
